# Claudio 口味档案

当前口味来自用户提供的网易云歌单截图。

## 偏好

- 华语流行、独立流行、轻 R&B。
- 喜欢 Live 的人声质感，但不需要吵闹的现场氛围。
- 偏雨天、夜晚、城市感、细腻情绪。
- 可以有失落感，但不要过分煽情。
- 歌曲应当能留在房间里，不抢走所有注意力。

## 默认避开

- 深夜时避开过亮、过吵、强鼓点的歌。
- 低落时不强行播放打鸡血的歌。
- 写代码时减少歌词密度和大段情绪爆发。

## 场景倾向

- 夜晚：陈奕迅、曹方、丁世光、告五人、吴炳文。
- 写代码：陶喆、丁世光、曹方。
- Emo 但别太丧：林俊杰、金玟岐、唐俗救世Vulgar Savior、颜人中。
- 睡前：就是哈比、吴炳文、曹方。
