# Loop 01 验收记录

日期：2026-06-20

## 已完成

- 新增 v0.2 Fastify 配置模块 `apps/server/src/config.ts`。
- v0.2 后端提供 `/api/config/status`。
- v0.1 兼容服务 `server.js` 也提供 `/api/config/status`，便于在依赖未补齐前验证。
- 当前前端页面新增 Providers 诊断面板。
- Provider 未配置时显示 `missing` 或 `fallback`，不伪装成真实服务已连接。

## 验收命令

```powershell
node --check server.js
node --check public\app.js
```

缺配置场景：

```powershell
PORT=8092 node server.js
GET http://127.0.0.1:8092/api/config/status
```

结果：

- `local-music`：`missing`
- `deepseek`：`missing`
- `netease`：`missing`
- `tts`：`fallback`
- `asr`：`fallback`
- `weather`：`missing`
- `calendar`：`missing`
- `upnp`：`missing`

有配置场景：

```powershell
LOCAL_MUSIC_DIR=D:\Music
DEEPSEEK_API_KEY=placeholder
NETEASE_PLAYLIST_ID=7637838720
```

结果：

- `local-music`：`ready`
- `deepseek`：`ready`
- `netease`：`ready`
