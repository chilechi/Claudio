# v0.2 Loop 08 验收：时段电台与长期口味

## 本轮范围

- 新增 `radio/routines.js`：
  - 按小时映射上午醒神、午休、会议间歇、运动、晚间、夜尾。
  - 按真实歌库标签、喜欢记录、跳过记录生成口味画像。
  - 推荐队列不随机，不用假数据补齐。
- 新增 API：
  - `GET /api/radio/today`
  - `GET /api/radio/today?hour=9`
  - `GET /api/taste/profile`
  - `GET /api/providers/status`
- 前端 TODAY PLAN 和 TASTE 面板改为接口驱动。
