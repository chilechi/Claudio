## 验收方式

使用临时数据库路径，不提交数据库文件：

```powershell
$env:DATABASE_PATH="data/test-claudio.sqlite"
npm run db:init
npm run db:import-v1
```

验收点：

- schema 可执行。
- `data/library.json` 可导入 tracks/playlists。
- `data/state.json` 可导入 preferences/memories/play_events。
- 数据库文件被 `.gitignore` 忽略，不进入 Git。

## 注意

`node:sqlite` 在当前 Node 版本可用，但仍显示 ExperimentalWarning。v0.2 先使用它避免原生依赖安装风险；后续如需稳定库，可替换为 `better-sqlite3` 或 `sqlite`。
